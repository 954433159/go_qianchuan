## 常见问题

### Q1: 页面显示404
**解决方案:**
- 检查URL是否正确
- 确认路由已在App.tsx中配置
- 尝试添加 `?preview=true` 参数跳过登录

### Q2: 侧边栏没有"账户管理"菜单
**解决方案:**
- 检查 `frontend/src/config/routes.ts`
- 确认 `mainNavigation` 中包含账户管理配置
- 清除缓存并重新加载

### Q3: 某些组件不显示
**解决方案:**
- 检查控制台错误
- 确认组件导入路径正确
- 运行 `npm run type-check` 检查类型错误

### Q4: 图表不显示
**解决方案:**
- 确认安装了recharts: `npm list recharts`
- 检查数据格式是否正确
- 查看控制台是否有recharts相关错误

### Q5: 开发服务器无法启动
**解决方案:**
```bash
# 清除node_modules重新安装
rm -rf node_modules package-lock.json
npm install

# 检查端口占用
lsof -ti:5173
kill -9 <PID>  # 如果端口被占用

# 重新启动
npm run dev
```
