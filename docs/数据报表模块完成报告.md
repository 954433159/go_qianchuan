## 🔧 技术细节

### TypeScript类型安全
- 所有API函数都有完整的参数和返回类型定义
- 接口继承：`SearchWordReport extends ReportData`
- 可选字段：`endTime?: string`
- 联合类型：`status: '直播中' | '已结束' | '预告中'`

### React Hooks使用
- `useState`: 状态管理（20+个状态变量）
- `useEffect`: 数据加载、依赖更新
- `useNavigate`: 路由跳转
- `useParams`: 动态路由参数提取
- `useToast`: 全局提示（自定义Hook）

### 性能优化
1. **懒加载**: 所有页面组件使用`React.lazy()`
2. **模拟延迟**: `await new Promise(resolve => setTimeout(resolve, 800))` 模拟真实API延迟
3. **条件渲染**: 只在数据加载完成后渲染图表
4. **事件委托**: 表格行点击使用单一处理函数

### 代码规范
- **命名规范**: 驼峰命名（函数、变量）、大驼峰（组件、类型）
- **导入顺序**: React → 第三方库 → 本地组件 → API → 工具函数 → 图标
- **注释**: 关键函数、模拟数据生成、真实API调用位置

## ⚠️ 注意事项

### 1. 依赖版本
确保已安装：
```json
{
  "@tremor/react": "^3.18.7",  // 包含recharts@2.15.4
  "lucide-react": "^0.344.0",
  "react-router-dom": "^6.22.0"
}
```

### 2. 图片占位符
当前使用`https://via.placeholder.com`，生产环境需替换为：
- 真实商品图片URL
- CDN托管的素材
- 或本地静态资源

### 3. 数据格式
模拟数据中的百分比格式：
- 后端返回：`"5.67%"` (字符串)
- 前端解析：`parseFloat("5.67%")` = 5.67

### 4. 时间处理
- `startTime`, `endTime`: ISO 8601字符串格式
- 显示：使用`new Date().toLocaleString('zh-CN')`
- 时区：注意服务器时区和本地时区差异

### 5. ID动态路由
直播间详情页使用`:id`动态参数：
```typescript
<Route path="/live-rooms/:id" element={...} />
```
通过`useParams()`获取：
```typescript
const { id } = useParams<{ id: string }>()
```
