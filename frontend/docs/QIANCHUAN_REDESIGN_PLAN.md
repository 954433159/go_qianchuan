# 千川直播带货场景 - UI/UX 重新设计方案

## 🎯 设计理念

**千川（巨量千川）是抖音电商的专属广告投放平台，核心场景是直播间带货和短视频带货。**

### 核心差异点

| 维度 | 巨量广告 (Ocean Engine) | 千川 (Qianchuan) |
|------|------------------------|------------------|
| **目标** | 品牌曝光、应用下载、线索收集 | **直播间推广、商品成交** |
| **核心要素** | 创意素材、落地页 | **直播间、商品、达人** |
| **关键指标** | CPM、CTR、转化成本 | **GMV、ROI、成交订单数** |
| **投放对象** | 通用广告创意 | **直播间、短视频** |
| **转化路径** | 点击→落地页→转化 | **直播间观看→商品点击→下单** |

## 🎨 设计特色

### 千川特色元素

1. **直播状态标识**
   - 🔴 直播中（红点+动画）
   - ⚫ 未开播（灰色）
   - 📅 预告（时间提醒）
   - ✅ 已结束（归档）

2. **GMV突出显示**
   - 使用大号数字
   - 使用金色/橙色等"成交"色调
   - 实时更新动画

3. **商品卡片**
   - 商品图片
   - 价格、库存、销量
   - 快速推广按钮
   - 数据预览（点击率、转化率）

4. **ROI指标**
   - 清晰的ROI数字
   - 趋势箭头
   - 目标达成度

5. **直播间缩略图**
   - 直播画面预览
   - 主播信息
   - 实时数据叠加
